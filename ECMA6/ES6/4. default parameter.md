# 4. default parameter (매개변수 기본값)

## 4-1. 소개

```js
const f = function (x, y, z) {
  x = x ? x : 4
  y = y || 5
  if (!z) {
    z = 6
  }
  console.log(x, y, z)
}
f(1)

f(0, null)
```

```js
const f = function (x, y, z) {
  x = x !== undefined ? x : 3
  y = typeof x !== "undefined" ? y : 4
  console.log(x, y)
}
f(0, null)
```

```js
const f = function (a = 1, b = 2, c = 3, d = 4, e = 5, f = 6) {
  console.log(a, b, c, d, e, f)
}
f(7, 0, "", false, null)
```

#### 4) `let` 선언과 동일한 효과

```js
const f = function (x = 1, y = 2 + x) {
  let z = y + 3
  x = 4
  console.log(x, y, z)
}
f()
```

#### 4-1) TDZ

```js
const multiply = function (x, y = x * 2) {
  console.log(x * y)
}
multiply(2, 3)
multiply(2)
```

```js
const multiply = function (x = y * 3, y) {
  console.log(x, y)
}
multiply(2, 3)
multiply(undefined, 2)
```

#### 4-2) 기본값으로 할당하고자 하는 값이 변수일 경우 주의사항

```js
let a = 10
let b = 20
function f (aa = a, b = b) {
  console.log(aa, b)
}
f(1, 2)
f(undefined, 2)
f(1)
f()
```

#### 4-3) arguments에도 영향을 줄까?

a()
